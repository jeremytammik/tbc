Join the Revit API Expert and Feedback Roundtable

I encourage all developers interested in the Revit API to join Boris Shafiro, me and the Revit development team in the Revit API expert roundtable and feedback session at Autodesk University this week, on Thursday, 19 November 2020, 15:00-16:00 o'clock Central European Time, CET.

To do so, please register for AU and navigate to the Idea Exchange feedback workshop #61:

Interact with a panel of Revit API experts who will answer your questions and discuss all relevant topics of your choice.

Share feedback on using Revit API: what's working and not, what's missing; (a) major painpoints and inefficiencies; (b) missing coverage/functionality; (c) priorities/wishes for future development.

Please note that prior .NET programming and Revit add-in development experience is required. This class is not suitable for beginners. Space is limited.
